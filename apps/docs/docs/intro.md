---
title: Giới thiệu
---

React Native Iconify cung cấp trải nghiệm biểu tượng thống nhất giữa iOS, Android và Web dựa trên hệ sinh thái Iconify.

Thư viện tập trung vào:

- **Dòng công cụ kiểu Monicon**: Loader CLI, plugin Metro, và Babel transform tùy chọn để giảm dung lượng bundle.
- **TypeScript đầy đủ**: Các icon được sinh ra kèm type union giúp IDE autocomplete chính xác.
- **Luồng phát hành rõ ràng**: Changeset, CI, và tài liệu chi tiết để cộng đồng dễ dàng đóng góp.

Đọc tiếp để cài đặt, cấu hình và tối ưu hoá cho dự án của bạn.
